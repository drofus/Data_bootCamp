Semana 9: An√°lisis estad√≠stico de datos#

üìë Estad√≠stica descriptiva#

La estad√≠stica descriptiva se encarga de resumir y describir las caracter√≠sticas principales de un conjunto de datos. Su objetivo es ofrecer una visi√≥n general del comportamiento de los datos antes de realizar an√°lisis m√°s complejos.

üéØ Variables continuas y discretas#

  • Discretas: Las variables discretas guardan valores especificos, normalmente son contables como por ejemplo: n√∫mero de hijos, cantidad de productos vendidos, etc.

  • Continuas: Las variables continuas guardan valores que se encuentren dentro de un rango o valores con mediciones precisas como por ejemplo: la altura, peso, temperatura, tiempo, etc.

# Variables discretas (n√∫mero de mascotas)
discretas = [0, 1, 2, 3, 1, 2, 3]

# Variables continuas (peso en kg)
continuas = [58.5, 61.2, 63.0, 59.7, 62.8, 64.1]

üìä Histogramas de frecuencia#

Un histograma de frecuencia muestra cu√°ntos datos caen dentro de intervalos espec√≠ficos. Son ideales para las variables continuas, ya que se puede utilizar facilmente para medir la frecuencia dentro de rangos.

Aspectos importantes:

  • Nos ayuda a visualizar la forma de la distribuci√≥n de datos.

  • Permite identificar patrones como asimetr√≠as, modas m√∫ltiples, o outliers.

  • Ayuda que los datos se agrupen en intervalos.

import numpy as np
import matplotlib.pyplot as plt

# Generar datos aleatorios
data = np.random.randint(165, 180, 50)

# Crear histograma de frecuencia
plt.hist(data, bins=10, edgecolor='black')
plt.title('Histograma de Frecuencia')
plt.xlabel('Rango de valores de estaturas (cm)')
plt.grid()
plt.ylabel('Frecuencia')

Text(0, 0.5, 'Frecuencia')

üìä Histogramas de densidad#

Un histograma de densidad es similar al de frecuencia, pero en lugar de mostrar el n√∫mero de datos, muestra la proporci√≥n de datos respecto al total.

Aspectos importantes:

  • Nos permite comparar datasets de diferentes tama√±os.

  • Se puede observar la ‚Äúprobabilidad‚Äù aproximada de caer en ciertos intervalos.

# Crear histograma de frecuencia
plt.hist(data, bins=10, density=True, edgecolor='black')
plt.title('Histograma de densidad')
plt.xlabel('Rango de valores de estaturas (cm)')
plt.grid()
plt.ylabel('Densidad')

Text(0, 0.5, 'Densidad')

üìê Medidas de posici√≥n y de variabilidad#

Las medidas de posici√≥n: las medidas de posici√≥n nos ayuda a caracterizar los valores ‚Äút√≠picos‚Äù que conocemos como:

  • Media: Promedio aritmetico.

  • Mediana: Valor que divide los datos en dos partes iguales.

  • Moda: El valor que m√°s se repite de los datos

Medida de variabilidad: Nos ayuda a observar que tan disperso est√°n nuestros datos:

  • Rango: Diferencia entre el valor m√°ximo y minimo.

  • Varianza: Medida promedio de cu√°nto se alejan los datos respecto a la media.

  • Desviaci√≥n estandar: Ra√≠z cuadrada de la varianza.

Estas medidas son esenciales para describir no solo el valor central, sino tambi√©n el comportamiento completo de los datos.

import scipy.stats as stats

# Media
media = int(np.mean(data))
# Mediana
mediana = int(np.median(data))
# Moda
moda = stats.mode(data)

print(f"Media: {media}, Mediana: {mediana}, Moda: {moda[0]}")

import pandas as pd

def get_box_plot_data(labels, bp):
    rows_list = []

    for i in range(len(labels)):
        dict1 = {}
        dict1['label'] = labels[i]
        dict1['lower_whisker'] = bp['whiskers'][i*2].get_ydata()[1]
        dict1['lower_quartile'] = bp['boxes'][i].get_ydata()[1]
        dict1['median'] = bp['medians'][i].get_ydata()[1]
        dict1['upper_quartile'] = bp['boxes'][i].get_ydata()[2]
        dict1['upper_whisker'] = bp['whiskers'][(i*2)+1].get_ydata()[1]
        rows_list.append(dict1)

    return pd.DataFrame(rows_list)

üîÉ Varianza#

La varianza cuantifica cu√°nto var√≠an los datos alrededor de la media.

  • Si la varianza es peque√±a, los datos est√°n cercanos a la media.

  • Si la varianza es grande, los datos est√°n m√°s dispersos.

Matem√°ticamente, es el promedio de las diferencias al cuadrado respecto a la media.

# Varianza
varianza = np.var(data)
print(f"Varianza: {varianza}")

Varianza: 18.72

üìè Desviaci√≥n est√°ndar y distribuci√≥n normal#

La desviaci√≥n est√°ndar es simplemente la ra√≠z cuadrada de la varianza. Tiene la ventaja de que sus unidades son las mismas que los datos originales. La distribuci√≥n normal es una de las distribuciones m√°s comunes en estad√≠stica:

  • Forma de ‚Äúcampana‚Äù sim√©trica.

  • La mayor√≠a de los valores est√°n cerca de la media. Conocer la distribuci√≥n normal es fundamental para t√©cnicas posteriores como inferencia estad√≠stica y machine learning.

import seaborn as sns
# Desviaci√≥n est√°ndar

üö® Datos sesgados y valores at√≠picos#

Los datos sesgados son una distribuci√≥n no sim√©trica, que no proporcionan una representaci√≥n precisa de un estudio. Existen dos tipos de sesgos: sesgo positivo y sesgo negativo.
El sesgo afecta medidas como la media y puede llevar a interpretaciones err√≥neas si no se detecta.

los valores at√≠picos son una observaci√≥n que es num√©ricamente distante del resto de los datos. Las estad√≠sticas derivadas de los conjuntos de datos que incluyen valores at√≠picos ser√°n frecuentemente enga√±osas.

#datos aleatorios
data = np.random.normal(165,5,1000)

# Detectar outliers usando el rango intercuart√≠lico
q1 = np.percentile(data, 25)
q3 = np.percentile(data, 75)
iqr = q3 - q1

limite_inferior = q1 - 1.5 * iqr
limite_superior = q3 + 1.5 * iqr

outliers = data[(data < limite_inferior) | (data > limite_superior)]
print(f"Valores at√≠picos: {outliers}")
